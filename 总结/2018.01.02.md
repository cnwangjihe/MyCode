### 差分约束系统  ###

简单来说就是一大堆二元不等式（且系数均为-1和1（均有一个）），可用矩阵表示，

如：
x0-x1<=a

x2-x1<=b

x1-x2<=c

—-         -—   —-    -—    —-   -—

|  1 -1  0  |   |  x0  |    |  a  |

|  0 -1  1  | * |  x1  | <= |  b  |

|  0  1 -1  |   |  x2  |    |  c  |

—-         -—   —-    -—    —-   -—

然后求满足上述不等式的某两个变量的差的最大值或最小值。
